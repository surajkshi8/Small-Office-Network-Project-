# Small-Office-Network-Project-
This project is a small office network made in Cisco Packet Tracer. It shows how to connect PCs, switches and a router so the devices can talk to each other. It includes basic IP setup, simple routing and a layout that anyone can follow. It’s meant for learning and for anyone who wants a clean example of a small office network.

# Small Office Network (Cisco Packet Tracer)

This project shows a basic small office network built in Cisco Packet Tracer. It includes a router, switches and a few PCs. The goal is to show how a simple office setup works and how the devices talk to each other.

# What’s inside
- Basic IP setup  
- Simple routing  
- PC to PC communication  
- A clear layout that’s easy to follow  

# How to use
1. Open the Packet Tracer file.
2. Check the IP settings on each device.
3. Try sending pings between the PCs.
4. Adjust the setup if you want to learn more.

# Requirements
- Cisco Packet Tracer 8.x or newer
